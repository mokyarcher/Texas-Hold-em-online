# Guizhou Poker - Product Brief

## Overview
**Guizhou Poker** is a LAN-based Texas Hold'em poker game platform designed for entertainment purposes. It supports multiplayer games on local networks with AI bots, making it perfect for friends and family gatherings.

---

## Key Features

### 🎮 Core Gameplay
- **Texas Hold'em Poker** - Classic poker rules with professional game flow
- **2-8 Players** - Flexible room sizes to fit your group
- **AI Bots** - Intelligent computer players when you need extra seats filled
- **Blind System** - Automatic small/big blind rotation with dealer button

### 🏠 Room Management
- **Create/Join Rooms** - Easy room creation with optional password protection
- **Multiple Blind Levels** - Choose from various blind settings (5/10, 10/20, 25/50, 50/100)
- **Room List** - Browse and join available games in real-time
- **Host Controls** - Room owners can manage and delete rooms

### 👤 User System
- **Account Registration** - Secure user accounts with password protection
- **Guest Login** - Quick play without registration
- **Avatar Selection** - 20+ avatars to personalize your profile
- **Nickname Customization** - Change your display name anytime
- **Chip System** - Virtual chips for betting (no real money involved)

### 🏆 Social Features
- **Leaderboard** - Compete for the top spot on the chip leaderboard
- **Online Status** - See who's online, in rooms, or in games
- **Player Profiles** - View stats and customize your profile

### ⚙️ Admin Features
- **User Management** - Admin can manage user accounts and chip balances
- **Room Cleanup** - Clear all rooms when needed
- **System Monitoring** - Track active games and players

## Game Flow

1. **Login/Register** - Create an account or play as guest
2. **Lobby** - Browse available rooms or create your own
3. **Room** - Wait for players, add bots if needed
4. **Game** - Play Texas Hold'em with betting rounds:
   - Pre-flop
   - Flop
   - Turn
   - River
   - Showdown
5. **Next Hand** - Stay in room to continue or return to lobby
