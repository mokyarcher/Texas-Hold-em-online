# 头像文件说明

## 📁 目录结构

```
backend/public/avatars/
├── avatar1.png
├── avatar2.png
├── avatar3.png
├── ...
└── avatar20.png
```

## 🎨 头像规格

- **尺寸**：50x50 像素
- **格式**：PNG（推荐）或 JPG
- **命名**：avatar1.png, avatar2.png, ..., avatar20.png

## 📝 使用说明

1. 将头像图片放入 `backend/public/avatars/` 目录
2. 图片命名格式：avatar[1-20].png
3. 前端通过文件名加载头像

## 🔧 预设头像

需要准备 20 个预设头像图片，命名为：
- avatar1.png
- avatar2.png
- avatar3.png
- ...
- avatar20.png

这些图片将作为默认头像供用户选择。
