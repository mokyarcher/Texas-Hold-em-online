# 德州扑克 - 座位排布规则

## 概述
本文档定义了不同人数桌的座位排布规则，确保布局美观合理。

## 座位编号规则
- **0号位**：始终为本人（当前用户）
- **1-N号位**：其他玩家按顺时针方向编号

## 各人数桌的座位位置

### 两人桌
- **0号位**：上中（本人）
- **1号位**：下中

### 三人桌
- **0号位**：左中（本人）
- **1号位**：右中
- **2号位**：下中

### 四人桌
- **0号位**：左中（本人）
- **1号位**：右中
- **2号位**：上中
- **3号位**：下中

### 五人桌
- **0号位**：中下（本人）
- **1号位**：右下
- **2号位**：右下
- **3号位**：左上
- **4号位**：左下

### 六人桌
- **0号位**：中下（本人）
- **1号位**：右下
- **2号位**：右下
- **3号位**：右上
- **4号位**：中上
- **5号位**：左上
- **6号位**：左下

### 七人桌
- **0号位**：中下（本人）
- **1号位**：右下
- **2号位**：右下
- **3号位**：右上
- **4号位**：中上靠右
- **5号位**：中上靠左
- **6号位**：左上
- **7号位**：左下

### 八人桌
- **0号位**：中下（本人）
- **1号位**：右下
- **2号位**：右下
- **3号位**：右上
- **4号位**：中上靠右
- **5号位**：中上靠左
- **6号位**：中上靠中
- **7号位**：左上
- **8号位**：左下

## 位置说明

### 方向定义
- **上**：屏幕上方区域（top: 10-20%）
- **下**：屏幕下方区域（bottom: 5%）
- **左**：屏幕左侧区域（left: 5-20%）
- **右**：屏幕右侧区域（right: 5-20%）
- **中**：屏幕中央区域（left: 50% 或 right: 50%）

### 位置组合
- **上中**：top + left: 50%
- **下中**：bottom + left: 50%
- **左中**：left + top: 50%
- **右中**：right + top: 50%
- **中上**：top + left: 50%
- **中下**：bottom + left: 50%
- **中上靠右**：top + left: 60%
- **中上靠左**：top + left: 40%
- **中上靠中**：top + left: 50%

## 设计原则
1. **本人始终在中央下方**：便于操作和观察
2. **对称分布**：左右对称，视觉平衡
3. **均匀间距**：避免重叠和拥挤
4. **圆形布局**：多人时形成圆形或椭圆分布
5. **美观优先**：确保视觉舒适，易于观察

## 实现细节
- 使用 CSS 的 `position: absolute` 定位
- 使用百分比（%）实现响应式布局
- 使用 `transform: translateX(-50%)` 实现水平居中
- 不同人数使用不同的配置数组
